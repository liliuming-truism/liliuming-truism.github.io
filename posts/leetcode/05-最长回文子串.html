最长回文子串

LiLiuMing8/27/25...About 2 minLeetCode算法

题目理解

最长回文子串（Longest Palindromic Substring）

给定一个字符串，找到其中最长的回文子串
回文串是指正读和反读都相同的字符串，如"aba"、"abba"

解题思路 中心扩展法

1、双中心尝试

对每个下标 i，尝试两种中心：[ (i,i) ] 表示奇数长度回文中心，[ (i, i+1) ] 表示偶数长度回文中心。分别向左右扩展。

2、expandAroundCenter 函数返回长度原因

循环结束时索引已经多走了一步（left 多减 1，right 多加 1），有效回文区间为 [left+1, right-1]，其长度为 [ (right-1) - (left+1) + 1 = right - left - 1 ]。

3 、更新区间公式来源

设当前中心为 i，得到的回文长度为 len。

奇偶统一公式：

左边界 [start = i - (len - 1)/2]

右边界 [end = i + len/2]

说明：

当 len 为奇数：[len = 2k+1]，则 start = i - k，end = i + k
当 len 为偶数：[len = 2k]，则 start = i - (2k-1)/2 = i - k + 0.5（实际是 i-k+1 在偶数中心情形中通过 i 的取值与对称处理达到正确落点），由于我们在调用时组合 (i,i+1) 的中心，使得该统一公式成立（在整数除法语言中需注意整除行为）。

4、时间复杂度

外层循环 [n] 次，每次最坏扩展 O(n)，故总体 [O(n^2)]。空间只用常量变量，故 [O(1)]。

public class Solution {
    public String longestPalindrome(String s) {
        if (s == null || s.length() < 1) return "";
        
        int start = 0, end = 0;
        
        for (int i = 0; i < s.length(); i++) {
            // 情况1：奇数长度回文串，以i为中心
            int len1 = expandAroundCenter(s, i, i);
            // 情况2：偶数长度回文串，以i和i+1之间为中心
            int len2 = expandAroundCenter(s, i, i + 1);
            
            int len = Math.max(len1, len2);
            
            // 如果找到更长的回文串，更新起始和结束位置
            if (len > end - start) {
                start = i - (len - 1) / 2;
                end = i + len / 2;
            }
        }
        
        return s.substring(start, end + 1);
    }
    
    // 辅助方法：从中心向两边扩展
    private int expandAroundCenter(String s, int left, int right) {
        while (left >= 0 && right < s.length() && s.charAt(left) == s.charAt(right)) {
            left--;
            right++;
        }
        return right - left - 1; // 返回回文串长度
    }
}
