字符串转整形

LiLiuMing9/6/25...About 3 minLeetCode算法

题目理解

请你来实现一个 myAtoi(string s) 函数，使其能将字符串转换成一个 32 位有符号整数。

函数 myAtoi(string s) 的算法如下：

空格：读入字符串并丢弃无用的前导空格（" "）
符号：检查下一个字符（假设还未到字符末尾）为 '-' 还是 '+'。如果两者都不存在，则假定结果为正。
转换：通过跳过前置零来读取该整数，直到遇到非数字字符或到达字符串的结尾。如果没有读取数字，则结果为0。
舍入：如果整数数超过 32 位有符号整数范围 [−231, 231 − 1] ，需要截断这个整数，使其保持在这个范围内。具体来说，小于 −231 的整数应该被舍入为 −231 ，大于 231 − 1 的整数应该被舍入为 231 − 1 。
返回整数作为最终结果。

上面是题目描述，通过题目描述可以发现，其实这个题目没有考察什么编程技巧，主要考察的是编程过程中边界条件的处理。
我们考虑这个题目的边界条件可以基于它的几个算法步骤进行设计

A 空格处理
全部是空格：" "
空字符串
前导空格 + 数字

B 符号处理
单正号："+8"
单负号："-8"
只有符号："+"/"-"

C 数字读取与终止条件
前导零："000001234"
只有零："0000"
数字后跟字母："123abc"
数字后面跟空格再有数字："123 456"

D 溢出与边界
恰好最大："2147483647"
恰好最小："-2147483648"
远超下限："-9999999999999"
远超上限："999999999999999"
前导零 + 溢出："000002147483648"

解题思路

第一步：使用jdk原生trim函数去除空格，去除完以后要判断字符串是否为空，如果为空，直接返回0
第二步：判断符号，这个时候如果是+/-符号，则需要判断去掉符号后的字符串是否为空，如果为空，直接返回0
第三步：移除前置0，这个地方可以使用递归实现，也可以while循环实现，需要注意的是，每次去除一个前置0的时候都要向上面一样判断字符串是否为空
第四步：遍历后面的字符，发现如果是属于目标数字类型的字符，则添加到StringBuilder上
第五步：将第四步的StringBuilder转数字，如果出现转换异常，说明发生了溢出，这个时候就根据第二步的符号判断返回整数的最大值还是最小值
如果没有异常，直接根据符号计算最终结果并返回

public class ConvertStringToInteger {

    private static final List<Character> nums = List.of('1', '2', '3', '4', '5', '6', '7', '8', '9', '0');

    public int myAtoi(String s) {
        s = s.trim();
        if (s.isEmpty()) {
            return 0;
        }
        boolean plus = true;
        if (s.charAt(0) == '-') {
            plus = false;
            s = s.substring(1);
        } else if (s.charAt(0) == '+') {
            s = s.substring(1);
        }
        if (s.isEmpty()) {
            return 0;
        }
        s = removeZero(s);
        if (s.isEmpty()) {
            return 0;
        }
        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < s.length(); i++) {
            char charI = s.charAt(i);
            if (nums.contains(charI)) {
                sb.append(charI);
            } else {
                break;
            }
        }
        if (sb.isEmpty()) {
            return 0;
        }
        try {
            int val = Integer.parseInt(sb.toString());
            if (!plus) {
                return -val;
            }
            return val;
        } catch (NumberFormatException e) {
            if (plus) {
                return Integer.MAX_VALUE;
            } else {
                return Integer.MIN_VALUE;
            }
        }
    }

    private String removeZero(String s) {
        if (s.isEmpty()) {
            return "";
        }
        if (s.charAt(0) == '0') {
            return removeZero(s.substring(1));
        }
        return s;
    }
}
