You can disable some function and layout on the page by setting the Frontmatter of the page.

<!-- more -->

This page is an demo that disables the following features:

- Navbar
- Sidebar
- Breadcrumb
- Page information
- Contributors
- Edit link
- Update time
- Prev/Next link
- Comment
- Footer
- Back to top button
